设置音量的方法也很简单，AudioManager提供了方法：
  1. public void setStreamVolume(int streamType, int index, int flags)

streamType为铃声类型，例如：AudioManager.STREAM_VOICE_CALL、AudioManager.STREAM_SYSTEM等，index为音量大小
falgs为标志位。


设置振动：

  2.                 long[] pattern = {150, 100}; // OFF/ON/OFF/ON...  
  3.                 mVibrator.vibrate(pattern, -1);

  1. //通话时设置静音System.out.println("isMicrophoneMute =" + audioManager.isMicrophoneMute());
